# Vibe Coders

Code with AI, no terminal fear.

Beautiful UI for AI-powered development. Build real projects with AI assistance without ever touching the terminal.

## Features

- **No Terminal Required** - Click buttons instead of typing commands
- **AI-Powered** - Generate commits, create projects, and build apps with AI
- **Dark Mode** - Beautiful light and dark themes
- **Full Code Access** - Own and control all code locally
- **Multiple Projects** - Manage Next.js apps, mobile apps, and more

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Tech Stack

- Next.js 16
- React 19.2
- TypeScript
- Tailwind CSS
- Shadcn UI
